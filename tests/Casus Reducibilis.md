# Casus Reducibilis (оператор Кардано)

- Вход: k, b; уравнение \(x^3 = kx + b\).
- Ключ: \(\Phi = (b/2)^2 - (k/3)^3\).
- Ветвление:
  - **Phi < 0** → casus irreducibilis:
    - Эвристика: поиск \(a,b\) таких, что \(a^2+b^2=k/3\), \(a^3-3ab^2=b/2\) → \(x=2a\).
    - Если не найдено → тригонометрическая форма (устойчивый реальный корень).
  - Иначе → классическая формула Кардано.

## ASCII-схема
Кубические → Phi
   ├─ Phi >= 0 → Кардано (комплексная форма) → x
   └─ Phi < 0  → Эвристика (a,b) ? → да → x=2a
                              └→ нет → тригонометрия → x
