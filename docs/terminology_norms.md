# Терминологические нормы ArcaLang  
### Канонический раздел дисциплины

Этот документ фиксирует ключевые термины ArcaLang, определяя их двойную природу:  
внешнюю (академическую) и внутреннюю (аркальную).

---

## 1. DSL

### Внешнее определение  
DSL = Domain‑Specific Language

### Внутреннее определение  
DSL = Dynamic Semantic Logic

### Статус  
Каноническая норма школы.

---

## 2. Phase

### Внешнее определение  
Состояние или этап вычислительного процесса.

### Внутреннее определение  
Фундаментальная единица хронотопной динамики ArcaLang.

### Статус  
Каноническая сущность языка.

---

## 3. Protoform

### Внешнее определение  
Абстрактный оператор или шаблон вычисления.

### Внутреннее определение  
Первичная семантическая форма ArcaLang — операторный организм.

### Статус  
Каноническая категория операторов.

## 4. Resonance

### Внешнее определение

### Внутреннее определение  
Динамика согласования структур ArcaLang.

### Статус  
Каноническая динамическая категория.

Структура, объединяющая время и пространство.

### Внутреннее определение  
Онтологический каркас ArcaLang — координатная система фазовой динамики.

### Статус  
Каноническая онтологическая сущность.
