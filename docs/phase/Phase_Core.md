# Phase.Core — базовые фазовые инварианты ArcaLang

Phase.Core — фундаментальный модуль фазовой онтологии ArcaLang.  
Он определяет минимальный набор инвариантов, необходимых для:

- фазовой геометрии  
- фазовой динамики  
- резонансных сетей  
- симуляций Qnd-ячейкок  
- фазовой памяти  

## Список операторов

### 1. `psi0_from_E(E)`
Потенциал локальной ячейки:


\[
\psi_0 = \sqrt[3]{1 - E^2}
\]



### 2. `psiglob_from_R_V(R_glob, V, V0)`
Глобальный потенциал пространства:


\[
\Psi_{\text{glob}} = \frac{4\pi R^3}{V - V_0}
\]



### 3. `deltaV_from_psi0_Psiglob(psi0, Psi_glob)`
Фазовая деформация объёма времени:


\[
\Delta V = \sqrt[3]{1 + \psi_0 / \Psi_{\text{glob}}}
\]



### 4. `phase_density(V, V0, R)`
Фазовая плотность:


\[
\rho_\phi = \frac{V - V_0}{4\pi R^3}
\]



### 5. `energy_density(E, R0)`
Энергетическая плотность:


\[
\rho_E = \frac{E}{4\pi R_0^3}
\]



### 6. `m_effective_from_E_R0_psi0(E, R0, psi0, v_over_c)`
Эффективная масса:


\[
m_0 = \frac{E \psi_0}{4\pi R_0^3 (v/C)^2}
\]



### 7. `resonance_ratio(Psi, psi_res)`
Резонансное отношение:


\[
\psi_{\text{new}} = \Psi / \psi_{\text{res}}
\]



### 8. `phase_index_n(R, V, V0)`
Индекс фазового заполнения:


\[
n = \frac{4\pi R^3}{V - V_0}
\]

## Назначение модуля

Phase.Core служит:

- догматическим фундаментом ArcaLang  
- источником истины для всех фазовых вычислений  
- базой для Phase.Geometry, Phase.Dynamics, Phase.Simulation  
- основой для фазовых карт и WEB-интерфейсов
