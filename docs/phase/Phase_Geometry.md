# Phase.Geometry — геометрические инварианты ArcaLang

Модуль **Phase.Geometry** определяет фундаментальные геометрические структуры,
которые используются в фазовой онтологии ArcaLang:

- сферические объёмы  
- тороидальные объёмы  
- глобальные объёмы пространства  
- нормированные геометрические плотности  

Эти операторы служат основой для фазовой динамики, симуляций и резонансных моделей.

---

## 📐 1. `sphere_volume(R)`
**Объём сферы**



\[
V = \frac{4}{3}\pi R^3
\]



**Назначение:**  
Базовая геометрическая форма для фазовых ячеек и глобальных структур.

**Ограничения:**  
- \( R > 0 \)

---

## 🌀 2. `torus_volume(R, r)`
**Объём тора**



\[
V = 2\pi^2 R r^2
\]



**Назначение:**  
Используется для моделирования тороидальных фазовых структур, включая Qnd‑кольца и резонансные оболочки.

**Ограничения:**  
- \( R > 0 \)  
- \( r > 0 \)
