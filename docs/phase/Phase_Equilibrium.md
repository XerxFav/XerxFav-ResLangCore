# Phase.Equilibrium — фазовое равновесие и устойчивые состояния

Фазовое равновесие — это состояние, при котором:

- энергия ячейки стабилизируется  
- память перестаёт адаптироваться  
- корреляции становятся устойчивыми  
- глобальный потенциал перераспределяется равномерно  

Модуль Phase.Equilibrium реализуется через оператор `equilibrium_check`
из Phase.Simulation.

---

## 🎯 Критерий равновесия



\[
\text{eq} = (|d\psi/dt| < \epsilon) \land (|dE| < \epsilon)
\]



Где:

- \( d\psi/dt \) — скорость изменения памяти  
- \( dE \) — изменение энергии  
- \( \epsilon \) — порог устойчивости  

---

## 📌 Интерпретация

### 1. **Стабильная память**


\[
|d\psi/dt| \to 0
\]



### 2. **Стабильная энергия**


\[
|dE| \to 0
\]



### 3. **Устойчивые корреляции**


\[
M_{ij} \ge \epsilon
\]



### 4. **Глобальная согласованность**


\[
\Psi / \psi_0 \approx E^2
\]



---

## 🧩 Роль в ArcaLang

Фазовое равновесие используется для:

- завершения симуляций  
- определения устойчивых фазовых структур  
- анализа фазовых переходов  
- построения фазовых карт  

**Phase.Equilibrium — критерий завершённости фазовой эволюции.**
