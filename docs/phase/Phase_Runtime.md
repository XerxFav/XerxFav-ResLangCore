# Phase.Runtime — вычислительное ядро фазовой модели ArcaLang

Модуль **Phase.Runtime** объединяет все фазовые операторы в единый
вычислительный слой. Он обеспечивает:

- выполнение фазовых шагов  
- обновление энергии и объёма  
- адаптацию памяти  
- проверку устойчивости  
- проверку равновесия  
- поддержку фазовых сетей  

Это — фундамент для интерпретатора фазовых симуляций.

---

## 🔧 1. Фазовый шаг (Phase Step)

Фазовый шаг объединяет:

1. обновление энергии  
2. обновление объёма  
3. адаптацию памяти  
4. проверку устойчивости  
5. проверку равновесия  



\[
\text{step} = \{E_{t+1}, \Delta V_{t+1}, \psi_{t+1}, M, eq\}
\]
